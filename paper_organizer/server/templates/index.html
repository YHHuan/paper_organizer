How to use

  • 1. Paste any PubMed URL, DOI, or bare PMID into the field above.
  • 2. Choose the reference manager you want the paper added to.
  • 3. Tap Add Paper. The server fetches metadata and imports it automatically.

Mobile Setup

To use Paper Organizer from your phone, your computer must be running the tunnel command: 

paper-organizer serve --tunnel

This starts a Cloudflare Tunnel and prints a public HTTPS URL. Open that URL on any device — no VPN or port-forwarding needed.

The tunnel URL changes each time the command is run. Share it fresh with lab members each session, or use a named tunnel for a stable URL.
